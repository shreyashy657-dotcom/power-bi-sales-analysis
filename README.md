# 📊 Sales Analysis Project (AI + Power BI)

## 🔍 Problem Statement

Analyze sales data to identify trends, top products, and reasons for sales decline.

## 🛠️ Tools Used

* Excel
* Power BI
* Generative AI

## 📊 Dashboard Features

* Total Sales, Profit, Quantity (KPI Cards)
* Monthly Sales Trend
* Top 5 Products
* Category-wise Analysis

## 🤖 AI Usage

Used Generative AI to generate insights and business recommendations.

## 📈 Key Insights

* Top products: Markers, Tables, Sofas
* Sales fluctuate monthly
* Office Supplies generate highest profit

## 🚀 Business Recommendations

* Focus on top-selling products
* Improve pricing strategy
* Run seasonal promotions
